## Assignment Description
This assignment demonstrates basic Dart programming concepts including lists, loops, functions, classes, asynchronous programming, and exception handling.

## Task 1: Number Analysis App
This program analyzes a list of integers and calculates the maximum value, minimum value, sum, and average using loops and functions.

## Task 2: Async Calculator App
This program implements a calculator using a class with arithmetic methods and demonstrates async/await, Future.delayed, and exception handling for divide-by-zero errors.
